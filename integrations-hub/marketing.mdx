## Mailchimp

**Installation:**
```bash
bp add mailchimp
```

**Description:**
Send mass email campaigns from within your workflows. Manage customers, campaigns, lists and more.

**Key Features:**
- **Email Campaigns:** Create and send marketing campaigns
- **List Management:** Manage subscriber lists and segments
- **Audience Management:** Add, update, and remove contacts
- **Campaign Analytics:** Track email performance
- **Templates:** Use and customize email templates
- **Automation:** Set up automated email workflows

**Use Cases:**
- Newsletter subscription via chatbot
- Lead capture and list building
- Campaign triggers from chat interactions
- Subscriber management
- Marketing automation workflows

**Actions:**
- Add subscribers to lists
- Create campaigns
- Send campaigns
- Update subscriber information
- Manage tags and segments
- Get campaign statistics

## Email Marketing Patterns

### Newsletter Subscription

```typescript
// User subscribes to newsletter
const email = event.payload.email
const name = event.state.user.name

// Add to Mailchimp list
await client.callAction({
  type: 'mailchimp:addSubscriber',
  input: {
    listId: 'your-list-id',
    email: email,
    firstName: name.split(' ')[0],
    status: 'subscribed',
    tags: ['chatbot-signup']
  }
})

// Send welcome email
await client.callAction({
  type: 'sendgrid:sendEmail',
  input: {
    to: email,
    subject: 'Welcome to our newsletter!',
    markdown: `# Welcome ${name}!\n\nThank you for subscribing.`
  }
})
```

### Transactional Email

```typescript
// Order confirmation email
await client.callAction({
  type: 'resend:sendEmail',
  input: {
    to: customerEmail,
    subject: `Order Confirmation #${orderId}`,
    markdown: `
# Order Confirmed!

**Order ID:** ${orderId}
**Total:** $${total}

## Items:
${items.map(item => `- ${item.name}: $${item.price}`).join('\n')}

Thank you for your order!
    `
  }
})
```

### Lead Nurturing Sequence

```typescript
// Add contact to Loops for drip campaign
await client.callAction({
  type: 'loops:createContact',
  input: {
    email: leadEmail,
    firstName: leadName,
    source: 'chatbot',
    customFields: {
      industry: leadIndustry,
      company_size: companySize
    }
  }
})

// Trigger email sequence
await client.callAction({
  type: 'loops:sendEvent',
  input: {
    email: leadEmail,
    eventName: 'lead_captured'
  }
})
```

## Best Practices

### Email Deliverability

1. **Authenticate your domain:**
   - Set up SPF records
   - Configure DKIM signing
   - Implement DMARC policy

2. **Maintain clean lists:**
   - Remove bounced emails
   - Honor unsubscribe requests
   - Validate email addresses

3. **Avoid spam triggers:**
   - Don't use excessive capitalization
   - Limit exclamation points
   - Include unsubscribe links
   - Maintain text/HTML balance

### Content Best Practices

1. **Personalization:**
   - Use recipient's name
   - Segment your audience
   - Tailor content to user behavior

2. **Mobile Optimization:**
   - Use responsive templates
   - Keep subject lines short
   - Use large, tappable buttons

3. **Clear CTAs:**
   - One primary call-to-action
   - Make buttons obvious
   - Use action-oriented text

### Compliance

1. **GDPR Compliance:**
   - Get explicit consent
   - Provide easy unsubscribe
   - Store consent records
   - Honor data deletion requests

2. **CAN-SPAM Compliance:**
   - Include physical address
   - Honor opt-out requests within 10 days
   - Use accurate subject lines
   - Label commercial emails

### Performance

1. **Timing:**
   - Send at optimal times
   - Consider time zones
   - Test send times

2. **Frequency:**
   - Don't over-email
   - Let users set preferences
   - Monitor unsubscribe rates

3. **Testing:**
   - A/B test subject lines
   - Test email templates
   - Preview on multiple clients

## Monitoring & Analytics

### Track key metrics:
- **Delivery rate:** Percentage of emails delivered
- **Open rate:** Percentage of emails opened
- **Click rate:** Percentage of links clicked
- **Unsubscribe rate:** Rate of opt-outs
- **Bounce rate:** Invalid or rejected emails

### Use webhooks for events:
- Email delivered
- Email opened
- Link clicked
- Email bounced
- Spam report
- Unsubscribe
